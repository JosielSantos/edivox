Esse é o Edivox, o editor de textos do [Sistema Dosvox][sist_dosvox].

Esse repositório serve para eu modificar e adicionar coisas que eu ache legais ao programa.

O repositório começou com o código fonte original, que inevitavelmente ficará desatualizado conforme o programa original sofrer atualizações.

#### Mudanças

* Adicionada opção de configuração para especificar o número de espaços ao apertar a tecla tab-  @JosielSantos
* Removidos arquivos de backup edtela2.pas e edarqnovo.pas - @thgcode
* Removida mensagem de configuração "aguarde, configurando" - @thgcode
* Adicionada opção de configuração para especificar se a tecla tab insere "\t" ou espaços - @thgcode
* Adicionada opção de configuração para determinar codificação ao salvar arquivo - @thgcode
* Adicionada opção de configuração para preservar últimas linhas em branco ao salvar o arquivo - @JosielSantos

[sist_dosvox]: http://intervox.nce.ufrj.br/dosvox/
